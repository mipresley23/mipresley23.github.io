Chipper

A Twitter inspired social media application.

Chipper allows users to signup and share what's going on in their life. They can see posts(chirps) from all users. Users can comment and like chirps and they have a profile page that displays their own chirps. Built using Javascript, React and Redux on the front end and using a Python, Flask, SQLAlchemy back end, useing a PostgreSQL database, and utilizing AWS S3 services for storing image uploads.

EarFruit

A Spotify inspired music streaming application.

EarFruit lets users stream their favorite songs. Users who decide to signup are given the option to upload songs and create playlists of their favorite songs. They also can view their profile page which lists any songs they have uploaded and their playlists. Built using Javascript, React, Redux, Python, Flask, and SQLAlchemy.

FloatBnB

An AirBnb inspired application with a watery twist.

FloatBnB lets users see listings for yacht rentals. Users who signup can create their own listings, book available listings, and see a profile page that lists listings they've made and rentals they've booked. Built with Javascript, React, Redux, Express.js, and Sequelize using a PostgreSQL database.
